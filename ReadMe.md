这款应用支持主流应用阅读内容（如微信，豆瓣，微博，知乎等）的收藏以及自定义收藏内容。

你看到的文章、你想要记的消息、你喜欢的图片（完善中）都可以收藏起来。

我的目标是它能够收集所有你想收集的东西。所以取名单字“集”。

如果在使用过程中有任何疑问或想法，欢迎告知。
